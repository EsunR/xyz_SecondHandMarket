## Data数据文件说明
在项目根目录下有Data文件夹，存放用户信息和商品信息
### 商品信息
路径：/Data/SecondHand/Item/item_id/  

说明：item_id为商品id命名的文件夹，遵循以下命名规则：
+ 商品封面：cover.jpg  
+ 商品展示中的其他图片：itemId_1.jgp, itemId_2.jgp, itemId_3.jgp ...  


### 用户信息
路径：/Data/User/user_id/

说明：uer_id为用户id命名的文件夹，遵循以下命名规则：
+ 用户头像：head.png
